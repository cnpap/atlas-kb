# 财务部职责

财务部负责预算管理、付款审核、费用报销、会计核算、税务申报、发票管理、资金计划、往来对账和财务结账。所有与资金支出、收入确认、税票合规和预算执行相关的事项，均由财务部主导处理，并对口径一致性与单据完整性负责。

典型受理事项包括：员工报销审核、供应商付款申请、项目成本归集、银行账户维护、增值税发票开具与认证、月度结账安排、预算追加申请、差旅借款核销以及收入确认口径咨询。若付款事项已经过业务审批但单据不完整，财务部有权退回并要求补齐附件后重新提交。

财务部不负责劳动合同签署、不负责合同条款法律审查、不负责供应商寻源与比价，也不负责客户培训排班。若事项涉及采购下单，应由采购部完成供应商比选；若合同中存在违约责任、赔偿条款或争议解决机制，需要法务部先出具审阅意见；若报销流程涉及人员入离职状态确认，则由人力资源部协助核实。

当有人提出“报销单应该交给哪个部门”“供应商付款资料由谁审核”“预算超标后找谁确认”时，应优先指向财务部，再由财务部决定是否需要采购、法务或业务负责人共同会签。

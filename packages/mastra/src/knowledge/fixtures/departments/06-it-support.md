# IT支持部职责

IT支持部负责终端设备开通、办公账号配置、常规软件安装、打印与网络接入支持、会议设备调试、终端故障排查和日常 IT 服务台响应。凡是与员工日常使用的电脑、显示器、打印机、邮箱、企业通信工具和办公网络可用性直接相关的问题，都应由 IT支持部优先受理。

典型受理事项包括：新员工电脑发放与初始化、邮箱账号开通、VPN 客户端安装、打印故障排查、会议室设备调试、办公软件升级、密码重置、普通网络异常定位以及终端系统性能问题处理。IT支持部在处理时需要记录故障现象、影响范围和恢复结果，并对重复故障给出标准解决方案。

IT支持部不负责合同条款审阅、不负责付款审批、不负责社保开户，也不单独主导信息安全事件取证。若电脑问题表现为中毒、钓鱼邮件点击、恶意程序扩散、敏感数据泄露或账号被盗，应立即转信息安全部主导处置，IT支持部配合完成终端隔离、镜像留存和恢复工作。

当有人问“电脑装软件找谁”“邮箱开通和密码重置归哪个部门”“会议室投屏问题谁来处理”时，应优先路由到 IT支持部；但若问题已经上升为安全事件，应同步通知信息安全部。

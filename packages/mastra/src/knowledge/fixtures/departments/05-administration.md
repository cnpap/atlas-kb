# 行政部职责

行政部负责办公环境维护、工位与会议室管理、固定资产领用协调、门禁与工牌办理、差旅与访客接待支持、办公用品统筹以及后勤服务供应商对接。凡是与办公场地、日常后勤、行政资源分配和现场支持相关的事项，原则上由行政部受理。

典型受理事项包括：新员工工位安排、门禁权限开通、工牌制作、会议室预留规则解释、办公用品领用、快递收发协调、来访接待、搬迁工位调整以及车辆与差旅行政支持。行政部需要保证现场资源使用秩序，并和人力资源部、IT支持部联动做好新员工入职准备。

行政部不负责劳动合同和社保手续、不负责电脑系统故障排查、不负责合同条款审阅，也不负责预算付款。若事项涉及“新员工入职当天的工位、门禁、工牌和办公用品”，行政部负责现场保障，但社保开户和劳动关系文件仍由人力资源部负责；若涉及电脑账号、邮箱或设备软件安装，则转 IT支持部处理。

当有人询问“门禁卡找哪个部门”“会议室与工位谁来安排”“访客接待由谁支持”时，应优先路由到行政部，再由行政部判断是否需要与人力资源部或 IT支持部协同。
